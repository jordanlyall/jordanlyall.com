Wallet-based gallery viewer for Art Blocks collections. Pulls from the Hasura API, resolves ENS names, aggregates linked wallets via Art Blocks profiles, and generates dynamic OG images.

Built because viewing generative art should feel like walking through a gallery -- not scrolling a token list.
